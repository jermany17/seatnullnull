## SEAT NULL NULL: AI를 활용한 교내 학습 공간 포화도 정보 제공 서비스

AI를 활용하여 교내 학습 공간에 대한 실시간 좌석 정보와 예측 포화도 정보를 제공하는 웹 애플리케이션

### 역할
Frontend

### 사용 기술  
React  

### 구현 기능  
메인 화면 & 장소 선택 화면, 실시간 좌석 정보 제공 화면, 예측 포화도 정보 제공 화면
